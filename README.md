# 💰 Tip Time App

Tip Time is a simple and elegant Android application that helps users quickly calculate tips based on service cost and tip percentage. It is built using **Kotlin** and **Jetpack Compose** as part of Android development practice.


## 📱 Features

- 💵 Enter service cost
- 📊 Select tip percentage (20%, 18%, 15% or custom)
- 🔄 Option to round up tip
- ⚡ Instant tip calculation
- 🎨 Clean and responsive UI using Jetpack Compose
- 📱 Supports modern Android devices


## 🛠️ Tech Stack

- **Language:** Kotlin
- **UI Toolkit:** Jetpack Compose
- **IDE:** Android Studio
- **Minimum SDK:** 21+


## 📂 Project Structure

```
TipTime/
│
├── MainActivity.kt
├── ui/
│   ├── components/
│   └── theme/
└── AndroidManifest.xml
```


## 🧮 How It Works

1. User enters the cost of service.
2. User selects the desired tip percentage.
3. Optionally enables "Round Up Tip".
4. App calculates and displays the final tip amount instantly.
